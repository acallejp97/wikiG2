Como instalar Proxmox Lo instalamos tal que cualquier sistema Linux: Escogemos el idioma, el teclado y la zona horaria Ponemos una contraseña y un email ponemos la configuracion de red, la IP y el Gateway. Reiniciamos y ya tendremos la instalacion hecha. Como acceder a Proxmox: Para acceder a Proxmox usamos la IP que pusimos en la configuración de red de Proxmox y la ponemos en el navagador junto con el puerto 8006 y https://. La interfaz de Proxmox nos pedirá el nombre de usuario y contraseña y una vez introducido podremos acceder.
